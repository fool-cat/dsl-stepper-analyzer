# DSL 步进电机分析工具使用说明

本目录用于直接给 Windows 用户使用。

主要文件：

- `dsl_stepper_speed_gui_win.exe`
  Windows 图形版主程序，双击即可运行。
- `往返.dsl`
  示例 `.dsl` 文件。
- `往返_电机_1_speed.html`
  示例分析结果。

## 使用方式

有两种常用启动方式。

### 方式 1：把 `.dsl` 文件拖到 exe 上

1. 选中一个或多个 `.dsl` 文件
2. 直接拖到 `dsl_stepper_speed_gui_win.exe` 上
3. 程序会自动启动，并打开浏览器工作台
4. 被拖入的 `.dsl` 会自动加入待分析列表

这种方式适合已经知道要分析哪些文件时直接打开。

### 方式 2：先打开 exe，再选择 `.dsl`

1. 双击 `dsl_stepper_speed_gui_win.exe`
2. 程序会自动打开浏览器工作台
3. 在页面中点击 `添加 DSL 文件`
4. 选择一个或多个 `.dsl` 文件后开始配置

这种方式适合先打开工具，再慢慢添加文件和调整配置。

## 页面里的操作流程

每个 `.dsl` 文件都会显示一张配置卡片，可以在里面设置一组或多组信号。

推荐按下面顺序操作：

1. 添加 `.dsl` 文件
2. 确认当前文件的信号组
3. 为每组选择 `DIR` 和 `STEP` 通道
4. 设置 `DIR 极性`
5. 设置 `STEP 边沿`
6. 点击 `生成/刷新对比视图`
7. 在页面下方直接查看结果，或点击 `打开 HTML`

## 各项配置说明

### 电机名

- 用来区分当前信号组的名称
- 会反映到生成的 HTML 文件名中
- 例如 `电机 1`、`X轴`、`右电机`

### DIR

- 选择方向信号所在通道，例如 `CH0`
- 如果没有方向信号，可选择 `未使用DIR（默认正向）`
- 选择 `未使用DIR` 后，程序会把所有运动都按正方向解析

### STEP

- 选择步进脉冲所在通道，例如 `CH1`
- `DIR` 和 `STEP` 不能选择成同一个通道

### DIR 极性

只有使用了 `DIR` 时才需要关注这个选项。

- `高电平为正方向`
  表示 `DIR=1` 时按正方向计算
- `低电平为正方向`
  表示 `DIR=0` 时按正方向计算

如果不确定，就按实际驱动器定义来选。

### STEP 边沿

用于指定以 `STEP` 的哪个边沿作为有效计步时刻。

- `上升沿计步`
- `下降沿计步`

如果你的驱动器或抓包约定是以上升沿有效，就选 `上升沿计步`；以下降沿有效，就选 `下降沿计步`。

## 多文件和多组信号

程序支持同时分析多个 `.dsl` 文件，也支持在同一个文件里配置多组信号。

常见用法：

- 多个 `.dsl` 同时加入页面做对比
- 一个 `.dsl` 文件里点 `新增信号组`，分别分析不同电机或不同通道组合

页面上的常用按钮：

- `添加 DSL 文件`
  继续加入新的 `.dsl`
- `新增信号组`
  在当前文件下增加一组新的 `DIR/STEP` 配置
- `删组`
  删除当前信号组
- `移除文件`
  把该 `.dsl` 从当前页面移除
- `生成/刷新对比视图`
  按当前配置重新生成结果
- `打开 HTML`
  单独打开某一组的分析页面
- `退出程序`
  关闭当前工作台

## 输出结果

生成结果为 HTML 页面，当前版本不生成 CSV。

HTML 中可查看：

- 速度曲线
- 加速度曲线
- 加加速度曲线
- 距离曲线

输出位置规则：

- 如果 `.dsl` 是本地文件路径加载的，HTML 会生成在原 `.dsl` 所在目录
- 文件名通常类似：
  `原文件名_电机名_speed.html`
- 如果是通过页面上传方式加入的文件，HTML 会放在程序临时目录中，但可以直接在页面里点击 `打开 HTML`

## 注意事项

- 建议使用 Windows 默认浏览器打开程序自动弹出的页面
- 浏览器工作台只在本机 `127.0.0.1` 打开，不会对外提供服务
- 程序默认只生成 HTML，不再生成 CSV
- 不再需要手工输入滤波窗口，程序默认使用自动稳健滤波
- 长时间不操作后，程序会自动退出

## 快速示例

可以直接拿本目录里的示例文件测试：

1. 把 `往返.dsl` 拖到 `dsl_stepper_speed_gui_win.exe`
2. 在页面里确认 `DIR`、`STEP`、`DIR 极性`、`STEP 边沿`
3. 点击 `生成/刷新对比视图`
4. 查看生成结果

如果只是单电机常见接法，可先尝试：

- `DIR = CH0`
- `STEP = CH1`
- `DIR 极性 = 高电平为正方向`
- `STEP 边沿 = 上升沿计步`

如果结果方向相反或计步时刻不对，再切换 `DIR 极性` 或 `STEP 边沿` 重新生成即可。
